# 🎯 Team Generator & Random Name Picker

A simple yet practical web app built with JavaScript that allows users to generate teams or randomly select a name from a list.

---

## 🚀 Features

* 🧑‍🤝‍🧑 Generate teams from a list of names
* 🎲 Randomly pick a name
* ✍️ Dynamic input handling (textarea support)
* 🔄 Reset functionality
* ⚡ Real-time interaction using DOM manipulation

---

## 🛠️ Built With

* HTML
* CSS (Flexbox)
* JavaScript (ES6)

---

## 🧠 What I Learned

* Working with arrays (`split`, `map`, `filter`)
* DOM manipulation and event handling

## 📂 How It Works

1. Enter names (one per line)
2. Select action:

   * Team Generator
   * Name Picker
3. Click **Submit**
4. View results instantly
